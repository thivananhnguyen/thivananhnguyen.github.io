Hotel Hypnos - Projet Web

Projet de gestion des réservations d'hôtel avec une intégration de diverses technologies modernes.

Technologies Utilisées

Le projet Hotel Hypnos utilise les technologies suivantes pour assurer une gestion fluide et sécurisée des réservations d'hôtels :

Environnement de Développement :

  • Docker : Environnement de développement isolé et déployable via Docker et Docker Compose
  • GitHub Actions : Intégration continue et déploiement automatisé
  • Git et GitHub : Versioning pour le contrôle de version et la collaboration

Backend :

  • Framework : Symfony 7
  • Langage : PHP 8.2
  • Gestion de Base de Données : PostgreSQL via Doctrine ORM
  • Tests : PHPUnit, Postman, MailDev
  • Gestion d'Images : Intégration de Firebase pour le stockage des images

Frontend :

  • Langages : HTML5, CSS3, JavaScript
  • Framework : Bootstrap pour la conception responsive

Serveur :

  • Nginx : avec HTTPS

Gestion du Projet et Conception :

  • Gestion de Projet : Trello pour l'organisation des tâches et la planification
  • Conception des Maquettes : Figma pour les prototypes visuels
  • Conception des données : Looping et Lucidchart

Fonctionnalités

Sécurité

Le projet intègre des mesures de sécurité avancées : HTTPS, validation des formulaires (inscription, connexion, contact), regex pour les champs sensibles, et gestion sécurisée des identifiants (.env, PgAdmin). Les mots de passe sont hachés et vérifiés par email (Maildev). Un contrôle d'accès basé sur les rôles (admin, manager, user) est mis en place, avec un pare-feu, protection CSRF et limitation des tentatives de connexion. La redirection et l'affichage des messages d'erreur sont optimisés pour une expérience utilisateur sécurisée.
